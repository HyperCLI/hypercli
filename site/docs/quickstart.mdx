---
title: Quickstart
description: Get started with HyperCLI
---

## 1. Install

```bash
pip install hypercli-sdk hypercli-cli
```

## 2. Configure

```bash
hyper configure
```

Or set environment variable:

```bash
export HYPERCLI_API_KEY=your_key
```

## 3. Verify

```bash
hyper billing balance
```

## 4. Use

```bash
# Chat with LLM
hyper llm chat deepseek-v3.1 "Hello!"

# Launch GPU job
hyper jobs create nvidia/cuda:12.0 -g l40s -c "nvidia-smi"

# Monitor with TUI
hyper jobs create nvidia/cuda:12.0 -g l40s -c "python train.py" -f
```
