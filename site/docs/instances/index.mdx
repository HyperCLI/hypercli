## Overview

HyperCLI provides on-demand and spot GPU instances across multiple regions worldwide.

## Quick Start

```bash
# Create a job with L40S GPU
hyper jobs create nvidia/cuda:12.0 -g l40s -c "python train.py"

# Create with H100 GPU
hyper jobs create nvidia/cuda:12.0 -g h100 -c "torchrun --nproc_per_node=8 train.py" -n 8
```

## Pricing Model

- **Spot instances** (default): Lower cost, may be preempted
- **On-demand instances**: Higher cost, guaranteed availability

```bash
# Spot (default)
hyper jobs create nvidia/cuda:12.0 -g l40s --spot

# On-demand
hyper jobs create nvidia/cuda:12.0 -g l40s --on-demand
```

## API Endpoints

- `GET /instances/pricing` - Current GPU pricing
- `GET /instances/types` - Available GPU types
- `GET /instances/regions` - Available regions
