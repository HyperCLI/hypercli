## Get Balance

```python
from hypercli import HyperCLI

hyper = HyperCLI()
balance = client.billing.balance()

print(f"Total: ${balance.total}")
print(f"Available: ${balance.available}")
print(f"Rewards: ${balance.rewards}")
print(f"Paid: ${balance.paid}")
```

### Balance Object

| Field | Type | Description |
|-------|------|-------------|
| `total` | str | Total balance (USD) |
| `available` | str | Available after reservations |
| `rewards` | str | Rewards balance |
| `paid` | str | Paid balance |

## List Transactions

```python
transactions = client.billing.transactions(limit=50, page=1)

for tx in transactions:
    print(f"{tx.transaction_type}: ${tx.amount_usd} ({tx.status})")
```

### Transaction Object

| Field | Type | Description |
|-------|------|-------------|
| `id` | str | Transaction ID |
| `amount` | int | Amount in smallest units |
| `amount_usd` | float | Amount in USD |
| `transaction_type` | str | `top_up`, `job`, `llm`, `refund`, etc. |
| `status` | str | `PENDING`, `COMPLETED`, `FAILED` |
| `rewards` | bool | Whether from rewards balance |
| `job_id` | str | Associated job ID (if applicable) |
| `created_at` | str | Timestamp |
