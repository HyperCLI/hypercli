---
title: Gateway Client
description: Low-level TypeScript gateway client for OpenClaw pairing, config, chat, files, channels, sessions, and cron.
---

`GatewayClient` is the low-level WebSocket client for the OpenClaw protocol.

Use it when you want direct access to:

- config RPC
- raw events
- chat/session control
- workspace files
- channel login/status flows
- cron RPC

## Create

```ts
import { GatewayClient } from "@hypercli.com/sdk";

const gateway = new GatewayClient({
  url: "wss://my-agent.hyperclaw.app",
  gatewayToken: "...",
  deploymentId: "agent-id",
  apiKey: "sk-...",
  apiBase: "https://api.hypercli.com/agents",
  autoApprovePairing: true,
});
```

Important options:

- `url`
- `gatewayToken`
- `deploymentId`
- `apiKey`
- `apiBase`
- `autoApprovePairing`
- `timeoutMs`
- `chatTimeoutMs`
- `clientId`
- `clientMode`
- `role`
- `scopes`
- `caps`

## Auth Model

This client does not use browser `/_jwt_auth` semantics.

It connects to the root agent host and authenticates at the OpenClaw protocol layer using:

- gateway token
- device identity
- pairing / cached device token

## Lifecycle

```ts
const off = gateway.onEvent((event) => {
  console.log(event);
});

await gateway.connect();
...
gateway.close();
off();
```

Lifecycle methods:

- `connect()`
- `start()`
- `close()`
- `stop()`
- `onEvent()`

## Readiness

```ts
await gateway.waitReady(300_000);
```

This is the recommended way to wait for a fresh agent after the pod reaches `RUNNING`.

## Config

```ts
await gateway.configGet();
await gateway.configSchema();
await gateway.configPatch({
  models: {
    providers: {
      openai: { apiKey: "sk-..." },
    },
  },
});
```

Methods:

- `configGet()`
- `configSchema()`
- `configPatch()`
- `configApply()`
- `configSet()`
- `status()`

## Chat And Sessions

Two patterns are useful:

### Raw RPC + event listeners

```ts
gateway.onEvent((event) => console.log(event));
await gateway.sendChat({ message: "Summarize this repository", sessionKey: "docs" });
```

### Higher-level history/session helpers

- `chatHistory()`
- `chatAbort()`
- `sessionsList()`
- `sessionsPreview()`
- `sessionsReset()`

## Files

```ts
await gateway.filesList("main");
await gateway.fileGet("main", "README.md");
await gateway.fileSet("main", "notes.txt", "hello");
```

## Channels

- `channelsStatus()`

## Models, Agents, Cron

- `modelsList()`
- `agentsList()`
- `cronList()`

- `cronRemove()`

## Approvals

- `execApprove()`
- `execDeny()`

## Recommendation

Most application code should not instantiate `GatewayClient` directly. Prefer:

- `OpenClawAgent.gateway()`
- `OpenClawAgent.connect()`
