## List Regions

```bash
curl https://api.hypercli.com/instances/regions \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Region Selection

Specify region when creating jobs:

```bash
# Create in Sweden
hyper instances launch nvidia/cuda:12.6.3-runtime-ubuntu22.04 -g l40s -r se -c "python train.py"

# Create in Virginia
hyper instances launch nvidia/cuda:12.6.3-runtime-ubuntu22.04 -g h100 -r va -c "python train.py"
```

If no region specified, the system selects based on availability and pricing.

## Region Considerations

- **Latency**: Choose regions close to your data sources
- **Availability**: GPU availability varies by region
- **Pricing**: Some regions may have different pricing
- **Data residency**: Consider compliance requirements
