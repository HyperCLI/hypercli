---
title: Gateway Client SDK
description: Use GatewayClient for OpenClaw config, chat, files, sessions, and cron over WebSocket.
---

`GatewayClient` is the low-level async WebSocket client for the OpenClaw gateway protocol. In most applications you will reach it through an `OpenClawAgent`.

## Create A Client

```python
from hypercli.gateway import GatewayClient

gw = GatewayClient(
    url="wss://openclaw-myagent.hypercli.com",
    token="jwt...",
    gateway_token="gateway-secret",
)
```

<Tip>
The easiest way to get a `GatewayClient` is via `OpenClawAgent.gateway()` or `OpenClawAgent.connect()`.
</Tip>

## Connection

```python
async with gw:
    print(gw.version, gw.protocol)
```

You can also call `connect()` and `close()` manually.

## Config

```python
config = await gw.config_get()
schema = await gw.config_schema()
await gw.config_patch({"models": {"providers": {"openai": {"apiKey": "sk-..."}}}})
await gw.config_set(config)
```

Use `config_patch()` for partial updates and `config_set()` to replace the full config.

## Chat And Sessions

```python
async for event in gw.chat_send("Explain this repository"):
    print(event)
```

Session helpers:

- `sessions_list()`
- `sessions_preview()`
- `sessions_reset()`
- `chat_history()`
- `chat_abort()`

## Files, Agents, And Cron

```python
await gw.files_list("main")
await gw.file_get("main", "README.md")
await gw.file_set("main", "notes.txt", "hello")

await gw.agents_list()
await gw.cron_list()
await gw.cron_add({"name": "nightly", "schedule": "0 2 * * *"})
await gw.cron_remove("job-id")
```

## Models And Status

```python
await gw.models_list()
await gw.status()
```
