## Main Client

```python
from hypercli import HyperCLI

client = HyperCLI(
    api_key=None,
    api_url=None,
    agent_api_key=None,
    agent_dev=False,
    agents_api_base_url=None,
    agents_ws_url=None,
)
```

## `client.agent`

Major methods:

- `plans()`
- `current_plan()`
- `subscriptions()`
- `subscription_summary()`
- `entitlements()`
- `cancel_subscription(subscription_id)`
- `models()`
- `discovery_health()`
- `discovery_config()`
- convenience inference helpers such as `chat()` and `openai()`

Important account types:

- `HyperAgentSubscription`
  recurring billing object with `cancel_at_period_end`, `can_cancel`, `plan_agent_tier`, and `slot_grants`
- `HyperAgentEntitlements`
  effective account state with `slot_inventory`, `pooled_tpd`, `active_entitlement_count`, and `billing_reset_at`
- `HyperAgentSubscriptionSummary`
  combined view with `entitlements`, `subscriptions`, `active_subscriptions`, and `billing_reset_at`

## Gateway

`GatewayClient` exposes:

- lifecycle: `connect()`, `close()`, `wait_ready()`
- config: `config_get()`, `config_schema()`, `config_patch()`, `config_set()`, `config_apply()`
- chat: `chat_send()`, `send_chat()`, `chat_history()`, `chat_abort()`
- sessions: `sessions_list()`, `sessions_preview()`, `sessions_reset()`
- files: `files_list()`, `file_get()`, `file_set()`
- channels: `channels_status()`, `channels_logout()`, `web_login_start()`, `web_login_wait()`
- models: `models_list()`
- cron: `cron_list()`, `cron_add()`, `cron_remove()`, `cron_run()`
- approvals: `exec_approvals_get()`, `exec_approvals_set()`, `exec_approval_resolve()`
- raw events: `on_event()`
