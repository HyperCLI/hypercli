## HyperCLI

Main SDK client.

```python
from hypercli import HyperCLI

client = HyperCLI(api_key=None, api_url=None)
```

- `client.billing`
- `client.jobs`
- `client.renders`
- `client.files`
- `client.user`

## X402Client

Use `X402Client` for pay-per-use x402 launches without a full account billing setup.

```python
from eth_account import Account
from hypercli import X402Client

acct = Account.from_key("0x...")
x402 = X402Client(api_url="https://api.hypercli.com")
```

### Methods

```python
x402.get_flow_catalog() -> list[FlowCatalogItem]
x402.get_flow_price(flow_type: str) -> float

x402.create_job(
  amount: float,
  account,
  image: str,
  command: str | None = None,
  gpu_type: str = "l40s",
  gpu_count: int = 1,
  region: str | None = None,
  interruptible: bool = True,
  ...
) -> X402JobLaunch

x402.create_flow(
  flow_type: str,
  amount: float,
  account,
  params: dict | None = None,
  notify_url: str | None = None,
) -> X402FlowCreate
```

## Errors

```python
from hypercli import APIError

try:
    client.jobs.get("invalid")
except APIError as e:
    print(e.status_code)
    print(e.detail)
```
