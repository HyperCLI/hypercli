## HyperCLI

```python
from hypercli import HyperCLI

client = HyperCLI(
    api_key=None,
    api_url=None,
    agent_api_key=None,
    agent_dev=False,
    agents_ws_url=None,
)
```

Namespaces:

- `client.billing`
- `client.jobs`
- `client.instances`
- `client.renders`
- `client.files`
- `client.keys`
- `client.user`
- `client.agent`
- `client.deployments`

## Errors

```python
from hypercli import APIError

try:
    client.jobs.get("invalid")
except APIError as e:
    print(e.status_code)
    print(e.detail)
```
