title: Billing

## Balance

```python
from hypercli import HyperCLI

client = HyperCLI()
balance = client.billing.balance()

print(f"Total: ${balance.total}")
print(f"Available: ${balance.available}")
print(f"Rewards: ${balance.rewards}")
print(f"Paid: ${balance.paid}")
```

### Balance Fields

| Field | Type | Description |
| --- | --- | --- |
| `total` | `str` | Total balance in USD |
| `available` | `str` | Spendable balance after reservations |
| `rewards` | `str` | Rewards balance |
| `paid` | `str` | Paid balance |

## Transactions

```python
transactions = client.billing.transactions(limit=50, page=1)

for tx in transactions:
    print(f"{tx.transaction_type}: ${tx.amount_usd} ({tx.status})")
```

Fetch one transaction directly:

```python
tx = client.billing.get_transaction("transaction_id")
```

### Transaction Fields

| Field | Type | Description |
| --- | --- | --- |
| `id` | `str` | Transaction ID |
| `amount` | `int` | Amount in smallest units |
| `amount_usd` | `float` | USD amount |
| `transaction_type` | `str` | `top_up`, `job`, `llm`, `refund`, and related values |
| `status` | `str` | `PENDING`, `COMPLETED`, `FAILED` |
| `rewards` | `bool` | Whether rewards balance was used |
| `job_id` | `str` | Associated job ID when present |
| `created_at` | `str` | Creation timestamp |
