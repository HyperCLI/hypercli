title: Billing
description: Check balances and inspect transaction history from the Python SDK.
---

`client.billing` covers wallet balances and ledger-style transaction history.

Recurring HyperClaw agent subscriptions are exposed separately under `client.agent`:

- `client.agent.subscriptions()`
  recurring billing objects, including `cancel_at_period_end`
- `client.agent.subscription_summary()`
  combined account view with active subscriptions plus effective entitlements
- `client.agent.entitlements()`
  effective current account state, including slot inventory, pooled inference, and `billing_reset_at`
- `client.agent.entitlement_instances()`
  concrete 1:1 entitlement grants with expiry, tags, and related agent bindings
- `client.agent.update_subscription(subscription_id, bundle)`
  update a recurring billing subscription to a new bundle

`update_subscription(...)` is the primary write path:

- upgrade
  apply a larger bundle
- downgrade
  schedule a smaller bundle for the next billing boundary
- cancel
  pass an empty bundle, which resolves to period-end cancellation

## Balance

```python
from hypercli import HyperCLI

client = HyperCLI()
balance = client.billing.balance()

print(f"Total: ${balance.total}")
print(f"Available: ${balance.available}")
print(f"Rewards: ${balance.rewards}")
print(f"Paid: ${balance.paid}")
```

### Balance Fields

| Field | Type | Description |
| --- | --- | --- |
| `total` | `str` | Total balance in USD |
| `available` | `str` | Spendable balance after reservations |
| `rewards` | `str` | Rewards balance |
| `paid` | `str` | Paid balance |

## Transactions

```python
transactions = client.billing.transactions(limit=50, page=1)

for tx in transactions:
    print(f"{tx.transaction_type}: ${tx.amount_usd} ({tx.status})")
```

Fetch one transaction directly:

```python
tx = client.billing.get_transaction("transaction_id")
```

### Transaction Fields

| Field | Type | Description |
| --- | --- | --- |
| `id` | `str` | Transaction ID |
| `amount` | `int` | Amount in smallest units |
| `amount_usd` | `float` | USD amount |
| `transaction_type` | `str` | `top_up`, `job`, `llm`, `refund`, and related values |
| `status` | `str` | `PENDING`, `COMPLETED`, `FAILED` |
| `rewards` | `bool` | Whether rewards balance was used |
| `job_id` | `str` | Associated job ID when present |
| `created_at` | `str` | Creation timestamp |
