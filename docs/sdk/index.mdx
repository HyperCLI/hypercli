## Install

```bash
pip install hypercli-sdk
```

Optional extras:

```bash
pip install "hypercli-sdk[comfyui]"
```

## Quick Start

```python
from hypercli import HyperCLI

client = HyperCLI()

balance = client.billing.balance()
print(balance.total)

job = client.jobs.create(
    image="nvidia/cuda:12.6.3-runtime-ubuntu22.04",
    gpu_type="l40s",
    command="nvidia-smi",
    dry_run=True,
)

print(job.job_id, job.state)
```

## Authentication And Config

Configuration is resolved in this order:

1. `HyperCLI(api_key="...")`
2. `HYPERCLI_API_KEY`
3. `~/.hypercli/config`

```python
client = HyperCLI(api_key="hyper_...")
```

## Client Namespaces

`client.billing`
- Balance and transaction history.

`client.jobs`
- GPU job lifecycle, logs, metrics, tokens, exec, and shell-related helpers.

`client.instances`
- GPU types, pricing, region metadata, and capacity.

`client.renders`
- Managed render jobs and high-level media flow helpers.

`client.files`
- File uploads for render inputs.

`client.keys`
- API key management.

`client.user`
- Current user metadata.

`client.agent`
- HyperClaw plan, key, and model discovery, plus convenience chat helpers.

`client.deployments`
- OpenClaw and generic agent runtime lifecycle, files, exec, logs, shells, and gateway sessions.

## Error Handling

```python
from hypercli import APIError, HyperCLI

try:
    client = HyperCLI()
    client.jobs.get("missing-id")
except APIError as e:
    print(e.status_code, e.detail)
```
