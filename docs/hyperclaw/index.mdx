## Overview

HyperClaw is HyperCLI's inference provider for agent workloads.

- Product: [https://hyperclaw.app](https://hyperclaw.app)
- Anthropic Messages API: `https://api.hyperclaw.app`
- OpenAI-compatible API: `https://api.hyperclaw.app/v1`
- Account/Billing API: `https://api.hyperclaw.app/api`

Use HyperClaw when you need production API keys, plan-based rate limits, and agent-ready chat access. For chat models, prefer the Anthropic Messages API. The OpenAI-compatible `/v1` endpoint remains available for compatibility.

## Checkout Options

- Stripe (card): `POST /api/stripe/{plan_id}`
- x402 (USDC): `POST /api/x402/{plan_id}`

x402 supports JWT and anonymous modes (account-linked or standalone key issuance).
