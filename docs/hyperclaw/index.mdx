## Overview

HyperClaw is HyperCLI's inference provider for agent workloads.

- Product: [https://hyperclaw.app](https://hyperclaw.app)
- Inference API: `https://api.hyperclaw.app/v1`
- Account/Billing API: `https://api.hyperclaw.app/api`

Use HyperClaw when you need production API keys, plan-based rate limits, and OpenAI-compatible model access.

## Checkout Options

- Stripe (card): `POST /api/stripe/{plan_id}`
- x402 (USDC): `POST /api/x402/{plan_id}`

x402 supports JWT and anonymous modes (account-linked or standalone key issuance).
