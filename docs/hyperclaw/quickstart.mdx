## Option 1: Web Dashboard (Easiest)

### 1. Create an Account

Sign up at [hyperclaw.app](https://hyperclaw.app) with email or wallet.

### 2. Subscribe to a Plan

From the [plans page](https://hyperclaw.app/dashboard/plans), choose a plan and pay with:

- **Credit card** — via Stripe, takes 30 seconds
- **USDC on Base** — via browser wallet (MetaMask, Coinbase Wallet, etc.)

### 3. Create an API Key

Go to [API Keys](https://hyperclaw.app/dashboard/keys) and click **Create Key**. Copy the key immediately — it won't be shown again.

### 4. Make a Request

```bash
export HYPERCLAW_API_KEY="sk-..."

curl https://api.hyperclaw.app/v1/chat/completions \
  -H "Authorization: Bearer $HYPERCLAW_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "kimi-k2.5",
    "messages": [{"role": "user", "content": "What is x402?"}]
  }'
```

Available models: `kimi-k2.5` (262K context) and `glm-5` (202K context, reasoning model).

Or with Python:

```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-...",
    base_url="https://api.hyperclaw.app/v1"
)

response = client.chat.completions.create(
    model="glm-5",  # or kimi-k2.5
    messages=[{"role": "user", "content": "What is x402?"}]
)
print(response.choices[0].message.content)
```

## Option 2: CLI

Subscribe with USDC via the HyperCLI:

```bash
# Install with wallet support
pip install 'hypercli-cli[wallet]'

# Create wallet
hyper wallet create

# Fund with USDC on Base (send to your wallet address)
hyper wallet balance

# Subscribe
hyper claw subscribe 1aiu

# Get your key
hyper claw status
```

<Note>
If you used the CLI, your key is saved at `~/.hypercli/claw-key.json`. No need to create one from the dashboard.
</Note>

See [CLI commands](/cli/commands/claw) for details.

## Option 3: x402-wallet (Headless)

For servers or CI/CD environments without a browser:

```bash
# Install x402-wallet
curl -L https://hyperclaw.app/binaries/x402-wallet -o x402-wallet
chmod +x x402-wallet

# Create and fund a wallet
./x402-wallet wallet-init --keystore
./x402-wallet config-set --network base

# Create payment signature and subscribe
./x402-wallet create-payment \
  --pay-to <RECIPIENT_ADDRESS> \
  --token 0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913 \
  --amount 35000000
```

See the [x402 wallet guide](/hyperclaw/x402-wallet) for the full flow.

## Verify Your Setup

### List Models

```bash
curl https://api.hyperclaw.app/v1/models \
  -H "Authorization: Bearer $HYPERCLAW_API_KEY"
```

### Check Your Plan

From the dashboard at [hyperclaw.app/dashboard](https://hyperclaw.app/dashboard), or via API:

```bash
curl https://api.hyperclaw.app/api/plans/current \
  -H "Authorization: Bearer <jwt-token>"
```
