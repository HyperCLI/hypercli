## What HyperClaw Is

HyperClaw is the dedicated inference provider for agentic workloads.

- Product site: [https://hypercli.com](https://hypercli.com)
- Inference API:
  - `https://api.hypercli.com` for `anthropic-messages` (recommended)
  - `https://api.hypercli.com/v1` for `openai-completions`
- Account/Billing API: `https://api.hypercli.com/agents`

Use HyperClaw when your agent needs LLM inference with API keys, subscription plans, and either Anthropic Messages or OpenAI-compatible clients.

## Why Agents Use It

- Anthropic Messages API for best agent/tool-calling behavior
- OpenAI-compatible API surface when your tool requires it
- Plan-based throughput controls (TPM/RPM)
- Checkout options:
  - Stripe (`/agents/stripe/{plan_id}`)
  - x402 USDC (`/agents/x402/{plan_id}`)

## Basic Agent Setup

```bash
export OPENAI_API_KEY="sk-..."
export OPENAI_BASE_URL="https://api.hypercli.com/v1"
```

Then select a HyperClaw model in your agent toolchain.

- For OpenClaw and agent setups, prefer `hyperclaw/kimi-k2.5-anthropic`.
- For direct Anthropic Messages usage, use `kimi-k2.5-anthropic` or `glm-5-anthropic`.
- For direct OpenAI-compatible usage, use `kimi-k2.5` or `glm-5`.
