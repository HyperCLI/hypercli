## Prerequisites

1. A HyperClaw key:

   ```bash
   hyper agent subscribe 1aiu
   ```

2. The CLI installed:

   ```bash
   pip install hypercli-cli
   ```

## Environment Variables

| Variable | Description | Default |
| --- | --- | --- |
| `HYPERCLAW_API_KEY` | HyperClaw API key used by `hyper agents` | Read from `~/.hypercli/agent-key.json` |
| `HYPERCLAW_API_BASE` | HyperClaw backend base URL | `https://api.hypercli.com` |
| `AGENTS_WS_URL` | Backend WebSocket override | Derived automatically |

## Create An Agent With The CLI

```bash
hyper agents create --name my-agent --size medium --wait
hyper agents status my-agent
hyper agents exec my-agent "echo hello world"
hyper agents shell my-agent
hyper agents chat my-agent
hyper agents stop my-agent --force
```

<Tip>
The CLI supports ID prefix matching from cached state in `~/.hypercli/agents.json`.
</Tip>

## Create An Agent With The Python SDK

```python
import time
from hypercli import HyperCLI

client = HyperCLI(api_key="...", agent_api_key="sk-...")

agent = client.deployments.create(name="my-agent", size="medium")
print(agent.id, agent.vnc_url)

while not agent.is_running:
    time.sleep(5)
    agent = client.deployments.get(agent.id)
    print(agent.state)

result = client.deployments.exec(agent, "echo hello world")
print(result.stdout)

client.deployments.file_write(agent, "/tmp/test.txt", "Hello from SDK!")
content = client.deployments.file_read(agent, "/tmp/test.txt")
print(content)

client.deployments.stop(agent.id)
```

## Use The Gateway

```python
import asyncio
from hypercli import HyperCLI

client = HyperCLI(api_key="...", agent_api_key="sk-...")

async def main():
    agent = client.deployments.get("your-agent-id")
    token = client.deployments.refresh_token(agent.id)
    agent.jwt_token = token["token"]

    async with agent.connect() as gw:
        await gw.config_patch({
            "models": {"providers": {"openai": {"apiKey": "sk-..."}}}
        })

        async for event in gw.chat_send("What files are in the workspace?"):
            print(event)

asyncio.run(main())
```
