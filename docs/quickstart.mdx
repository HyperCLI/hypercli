---
title: Quickstart
description: Get started with HyperCLI jobs, flows, HyperClaw plans, and OpenClaw agents.
---

## 1. Install

```bash
pip install hypercli-sdk hypercli-cli
```

## 2. Configure

```bash
hyper configure
```

## 3. Verify Account Access

```bash
hyper billing balance
hyper instances list
```

## 4. Launch A GPU Job

```bash
hyper instances launch nvidia/cuda:12.6.3-runtime-ubuntu22.04 -g l40s -c "nvidia-smi"
```

## 5. Run A Flow

```bash
hyper flow text-to-image "a cinematic landscape"
```

## 6. Optional: x402 Pay-Per-Use

```bash
hyper wallet create
hyper flow text-to-image "a cinematic landscape" --x402
```

## 7. HyperClaw

```bash
hyper agent plans
hyper agent config env
```

## 8. OpenClaw Agents

```bash
hyper agents create --name my-agent --size medium
hyper agents shell <agent_id>
```
