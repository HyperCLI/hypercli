## Inference Provider

HyperCLI inference is provided by **HyperClaw**.

- Product: [https://hypercli.com](https://hypercli.com)
- Inference endpoint: `https://api.hypercli.com/v1`
- Dashboard and plans: `https://hypercli.com`

## Quick Start

```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-...",
    base_url="https://api.hypercli.com/v1"
)

resp = client.chat.completions.create(
    model="kimi-k2.5",
    messages=[{"role": "user", "content": "hello"}],
)
print(resp.choices[0].message.content)
```
