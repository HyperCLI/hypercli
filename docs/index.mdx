HyperCLI is the unified control plane for GPU jobs, media flows, inference, and
agent runtimes.

Use it when you need to:

- launch raw GPU capacity on demand
- run image, video, speech, and transcription flows
- integrate against HyperClaw chat and embeddings endpoints
- deploy and operate OpenClaw desktop agents

## Install

```bash
pip install hypercli-sdk hypercli-cli
npm install @hypercli.com/sdk
```

## Configure

```bash
hyper configure
```

Or set the API key directly:

```bash
export HYPERCLI_API_KEY=hyper_...
```

## Quick Examples

### Launch A GPU Job

```bash
hyper instances launch nvidia/cuda:12.6.3-runtime-ubuntu22.04 -g l40s -c "nvidia-smi"
```

### Run A Flow

```bash
hyper flow text-to-image "a product hero image"
```

### x402 Pay-Per-Use

```bash
hyper flow text-to-image "a product hero image" --x402
```

### HyperClaw

```bash
hyper agent plans
hyper agent config env
```
