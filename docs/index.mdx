title: HyperCLI
description: GPU orchestration, media flows, and HyperClaw inference
---

HyperCLI provides:

- GPU job orchestration (`/api/jobs`)
- Media generation flows (`/api/flow/*`)
- Pay-per-use x402 launches for jobs/flows (`/api/x402/*`)
- Agent inference via HyperClaw (`https://hyperclaw.app`)

## Install

```bash
pip install hypercli-sdk hypercli-cli
```

## Configure

```bash
hyper configure
# or
export HYPERCLI_API_KEY=your_key
```

Get your API key at [console.hypercli.com](https://console.hypercli.com).

## Quick Examples

### Launch a GPU Job

```bash
hyper jobs create nvidia/cuda:12.0 -g l40s -c "python train.py" -f
```

### Run a Flow

```bash
hyper flow text-to-image "a product hero image"
```

### x402 Pay-Per-Use Flow

```bash
hyper flow text-to-image "a product hero image" --x402
```

### HyperClaw Inference

```bash
export HYPERCLI_API_URL="https://api.hyperclaw.app"
export HYPERCLI_API_KEY="sk-..."
hyper claw plans
hyper claw config env
```
