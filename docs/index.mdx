## Install

```bash
pip install hypercli-sdk hypercli-cli
npm install @hypercli.com/sdk
```

## Configure

```bash
hyper configure
```

Or set the API key directly:

```bash
export HYPERCLI_API_KEY=hyper_...
```

## Quick Examples

```bash
# Launch a GPU job
hyper instances launch nvidia/cuda:12.6.3-runtime-ubuntu22.04 -g l40s -c "nvidia-smi"

# Run a managed flow
hyper flow text-to-image "a product hero image"

# Discover HyperClaw plans
hyper agent plans
```
