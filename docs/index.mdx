---
title: HyperCLI
description: GPU orchestration, media flows, HyperClaw inference, and OpenClaw agents.
---

HyperCLI provides:

- GPU job orchestration through jobs and instance launches
- Managed media flows for image, video, speech, and transcription
- x402 pay-per-use launches for jobs and flows
- HyperClaw inference plans and model discovery
- OpenClaw and agent container deployments

## Install

```bash
pip install hypercli-sdk hypercli-cli
npm install @hypercli.com/sdk
```

## Configure

```bash
hyper configure
```

Or set the API key directly:

```bash
export HYPERCLI_API_KEY=hyper_...
```

## Quick Examples

### Launch A GPU Job

```bash
hyper instances launch nvidia/cuda:12.6.3-runtime-ubuntu22.04 -g l40s -c "nvidia-smi"
```

### Run A Flow

```bash
hyper flow text-to-image "a product hero image"
```

### x402 Pay-Per-Use

```bash
hyper flow text-to-image "a product hero image" --x402
```

### HyperClaw

```bash
hyper agent plans
hyper agent config env
```
