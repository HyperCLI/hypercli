Inference access limits (TPM/RPM) are controlled by your HyperClaw plan.

## Check Available Plans

```bash
curl https://api.hypercli.com/api/plans
```

## Check Your Current Plan

```bash
curl https://api.hypercli.com/api/plans/current \
  -H "Authorization: Bearer <jwt>"
```

## Notes

- Authenticated x402 purchases attach plan/subscription data to your account.
- Anonymous x402 purchases return standalone API keys.

See [Agents API](/agents/api) and [Inference for Agents](/agents/provider).
