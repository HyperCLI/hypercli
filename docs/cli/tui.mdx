---
title: TUI
description: Follow jobs with live logs and metrics from the terminal UI.
---

The CLI includes a full-screen job monitor driven by the `hyper jobs logs --tui` path.

## Launch The TUI

```bash
hyper jobs logs <job_id> --tui
```

Optional behavior:

```bash
hyper jobs logs <job_id> --tui --cancel-on-exit
```

## What It Shows

- current job state
- elapsed runtime
- hostname and websocket status
- CPU and memory usage
- per-GPU utilization, VRAM, temperature, and power
- live container logs

## Related Live Views

For a simpler metrics-only view:

```bash
hyper jobs metrics <job_id> --watch
```

For a plain streaming log view:

```bash
hyper jobs logs <job_id> --follow
```
