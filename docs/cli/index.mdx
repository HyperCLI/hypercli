## Install

```bash
pip install hypercli-cli
```

## Configure

```bash
hyper configure
```

Or set:

```bash
export HYPERCLI_API_KEY=your_key
```

## Common Commands

```bash
# Billing
hyper billing balance

# GPU jobs
hyper jobs create nvidia/cuda:12.0 -g l40s -c "nvidia-smi"
hyper jobs list
hyper jobs logs <job_id>

# Flows (recommended media API)
hyper flow text-to-image "a cinematic portrait"
hyper flow text-to-image "a cinematic portrait" --x402

# LLM via HyperClaw (set HYPERCLI_API_URL first)
export HYPERCLI_API_URL="https://api.hyperclaw.app"
hyper claw plans
hyper claw plans
hyper claw config env
```

## Notes

- `hyper renders create --x402` is not supported; use `hyper flow ... --x402`.
- `hyper claw *` commands are for HyperClaw plan checkout and key management.
