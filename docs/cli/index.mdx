## Install

```bash
pip install hypercli-cli
```

Optional extras:

```bash
pip install "hypercli-cli[wallet]"
pip install "hypercli-cli[comfyui]"
pip install "hypercli-cli[stt]"
```

HyperClaw:

- `hyper agent`
- `hyper agents`

## Split Between `hyper agent` And `hyper agents`

Use:

- `hyper agent`
  - account setup
  - plans
  - login / subscribe
  - provider config snippets
  - local transcription / voice / embed helpers
- `hyper agents`
  - actual agent lifecycle
  - files
  - exec
  - backend shell/log streams
  - gateway config/chat/session/files/cron

## Current Agent Route Model

The CLI now follows the same route defaults as the SDKs:

- root host = OpenClaw gateway / agent
- `desktop-<host>` = desktop
- there is no default `shell-<host>` route

So:

- `hyper agents shell` uses backend `/ws/shell/{agent_id}`
- `hyper agents logs --ws` uses backend `/ws/logs/{agent_id}`
- desktop opens on `desktop-<host>`
