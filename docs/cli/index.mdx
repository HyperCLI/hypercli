## Install

```bash
pip install hypercli-cli
```

Optional extras:

```bash
pip install "hypercli-cli[wallet]"
pip install "hypercli-cli[comfyui]"
pip install "hypercli-cli[stt]"
```

## Configure

```bash
hyper configure
```

Or set the main API key directly:

```bash
export HYPERCLI_API_KEY=hyper_...
```

## Command Groups

`hyper instances`
- Browse GPU inventory and launch jobs.

`hyper jobs`
- Inspect running jobs, stream logs, collect metrics, and open shells.

`hyper flow`
- Run the recommended media generation endpoints.

`hyper renders`
- Use the lower-level managed render API.

`hyper billing`
- Check account balance and transactions.

`hyper keys`
- Create, list, and disable API keys.

`hyper agent`
- Manage HyperClaw subscriptions and generate client configs.

`hyper agents`
- Manage OpenClaw or agent containers.

`hyper wallet`
- Create and fund the embedded wallet used for x402 payments.

`hyper comfyui`
- Run templated ComfyUI workflows on GPU jobs.

## Quick Examples

```bash
hyper instances list
hyper instances launch nvidia/cuda:12.6.3-runtime-ubuntu22.04 -g l40s -c "nvidia-smi"
hyper jobs logs <job_id> -f
hyper flow text-to-image "a product hero image"
hyper agent plans
hyper agents create --name docs-demo
```
