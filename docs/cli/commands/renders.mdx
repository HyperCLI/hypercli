---
title: hyper renders (legacy)
description: Legacy note for the removed top-level renders command.
---

`hyper renders` is no longer a live top-level CLI command.

Use:

- `hyper flow ...` for current media generation
- `client.renders` in the Python SDK
- `client.renders` in the TypeScript SDK

```bash
hyper flow text-to-image "a cat in space"
hyper flow text-to-video "a cat walking through snow"
hyper flow list
hyper flow get <render_id>
```

For managed render APIs in code, see:

- [Python renders SDK](/sdk/renders)
- [TypeScript renders SDK](/ts-sdk/renders)
