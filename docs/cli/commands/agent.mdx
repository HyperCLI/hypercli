title: hyper agent
description: Detailed reference for HyperClaw account, subscription, config, onboarding, and local audio commands.
---

`hyper agent` is the account and local-tooling surface for HyperClaw.

Use it for:

- onboarding
- wallet-based subscribe/login
- plan and model discovery
- provider config snippets
- local transcription
- voice and embedding helpers

## Commands

### `onboard`

Guided flow for:

- wallet creation
- funding
- plan selection
- subscription
- config generation
- verification

### `subscribe`

Subscribe to a HyperClaw plan through the x402 wallet flow.

```bash
hyper agent subscribe 1aiu
hyper agent subscribe 5aiu 100 --dev
```

### `status`

Show the saved HyperClaw key and current plan status.

### `plans`

List plan IDs, price, and limits.

```bash
hyper agent plans
hyper agent plans --dev
```

### `models`

List HyperClaw inference models.

```bash
hyper agent models
hyper agent models --json
```

### `login`

Wallet-based login that creates a user-bound HyperClaw API key and stores it locally.

### `config`

Generate config snippets for:

- `openclaw`
- `opencode`
- `env`

```bash
hyper agent config
hyper agent config openclaw --apply
hyper agent config env
```

### `openclaw-setup`

Patch `~/.openclaw/openclaw.json` directly using the saved HyperClaw key.

```bash
hyper agent openclaw-setup
hyper agent openclaw-setup --default
```

### `exec` and `shell`

Compatibility forwards into `hyper agents`.

### `transcribe`

Local faster-whisper transcription, no remote API required.

### `voice`

Voice generation helpers.

### `embed`

Embedding helpers against HyperClaw.

## Files Written By `hyper agent`

- `~/.hypercli/agent-key.json`
- optionally `~/.openclaw/openclaw.json` when using `openclaw-setup`

## Recommended Flow

1. `hyper agent plans`
2. `hyper agent subscribe 1aiu`
3. `hyper agent status`
4. `hyper agent openclaw-setup`
5. `hyper agents create ...`
