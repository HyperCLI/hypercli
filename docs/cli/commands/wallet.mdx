## Overview

The wallet is used for:

- HyperClaw subscriptions via `hyper agent subscribe`
- x402 pay-per-use flows via `hyper flow ... --x402`

Install wallet support with:

```bash
pip install "hypercli-cli[wallet]"
```

## create

```bash
hyper wallet create
```

Creates an encrypted keystore at `~/.hypercli/wallet.json`.

## address

```bash
hyper wallet address
```

Shows the public wallet address without unlocking the keystore.

## balance

```bash
hyper wallet balance
```

Checks the wallet's USDC balance on Base mainnet.

## Environment Variable

To skip repeated passphrase prompts:

```bash
export HYPERCLI_WALLET_PASSPHRASE="your-secure-passphrase"
```
