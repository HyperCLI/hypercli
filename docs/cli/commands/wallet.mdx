## Overview

The wallet is used for:

- HyperClaw subscriptions via `hyper agent subscribe`
- x402 pay-per-use flows via `hyper flow ... --x402`

Install wallet support with:

```bash
pip install "hypercli-cli[wallet]"
```

## create

```bash
hyper wallet create
```

Creates an encrypted keystore at `~/.hypercli/wallet.json`.

If you want a migration-friendly plaintext wallet file first:

```bash
hyper wallet create --no-passphrase
```

That writes a plaintext private-key wallet file at the same path. You can encrypt it later.

## encrypt

```bash
hyper wallet encrypt --passphrase "your-passphrase"
```

Encrypts a plaintext wallet file in-place.

Legacy encrypted keystores continue to load unchanged. This command only rewrites the file when you explicitly run it.

## decrypt

```bash
hyper wallet decrypt --passphrase "your-passphrase"
```

Decrypts an encrypted keystore into a plaintext wallet file in-place.

## address

```bash
hyper wallet address
```

Shows the public wallet address without unlocking the keystore.

## balance

```bash
hyper wallet balance
```

Checks the wallet's USDC balance on Base mainnet.

## Environment Variable

To skip repeated passphrase prompts:

```bash
export HYPERCLI_WALLET_PASSPHRASE="your-secure-passphrase"
```

This only applies when the wallet file is encrypted.
