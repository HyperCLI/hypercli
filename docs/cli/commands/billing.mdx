---
title: hyper billing
description: Billing commands
---

## balance

Get account balance.

```bash
hyper billing balance
hyper billing balance -o json
```

**Options:**
- `-o, --output` - Output format: `table` (default) or `json`

## transactions

List transactions.

```bash
hyper billing transactions
hyper billing transactions -n 50
hyper billing transactions -p 2
hyper billing transactions -o json
```

**Options:**
- `-n, --limit` - Number of transactions (default: 20)
- `-p, --page` - Page number (default: 1)
- `-o, --output` - Output format: `table` or `json`
