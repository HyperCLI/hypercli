Uses the OpenAI-compatible API at `https://api.hypercli.com/v1`.

## models

List available models.

```bash
hyper llm models
hyper llm models -o json
```

## chat

Chat completion.

```bash
# One-shot
hyper llm chat <model> "<prompt>"
hyper llm chat deepseek-v3.1 "Explain quantum computing"

# With system prompt
hyper llm chat deepseek-v3.1 "Write a haiku" -s "You are a poet"

# Interactive mode (omit prompt)
hyper llm chat deepseek-v3.1
```

**Arguments:**
- `model` - Model name (required)
- `prompt` - User prompt (omit for interactive mode)

**Options:**
- `-s, --system` - System message
- `-m, --max-tokens` - Max tokens (default: 4096)
- `-t, --temperature` - Temperature (default: 0.7)
- `--no-stream` - Disable streaming
- `-o, --output` - Output format: `text` or `json`

### Interactive Mode

When you omit the prompt, the CLI enters interactive chat mode:

```bash
hyper llm chat deepseek-v3.1

C3 Chat - Model: deepseek-v3.1
Type 'exit' or Ctrl+C to quit

You: Hello!
Assistant: Hi! How can I help you today?

You: exit
```
