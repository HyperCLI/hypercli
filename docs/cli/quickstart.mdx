---
title: CLI Quickstart
description: Get started with jobs, flows, x402, and HyperClaw from the CLI
---

## Install and Configure

```bash
pip install hypercli-cli
hyper configure
```

## Launch a GPU Job

```bash
hyper jobs create nvidia/cuda:12.0 -g l40s -c "nvidia-smi"
```

## Run a Flow

```bash
hyper flow text-to-image "a mountain lake at dawn"
```

## x402 Pay-Per-Use Flow

```bash
hyper flow text-to-image "a mountain lake at dawn" --x402
```

## HyperClaw Inference

```bash
export HYPERCLI_API_URL="https://api.hyperclaw.app"
export HYPERCLI_API_KEY="sk-..."
hyper claw plans
hyper claw config env
```

## HyperClaw Plan Checkout

```bash
hyper claw plans
hyper claw subscribe 1aiu
hyper claw status
```
