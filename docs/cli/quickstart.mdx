---
title: CLI Quickstart
description: Install the CLI, configure credentials, launch a GPU job, and run media flows.
---

## Install

```bash
pip install hypercli-cli
```

Add extras when you need them:

- wallet and x402: `pip install 'hypercli-cli[wallet]'`
- ComfyUI workflows: `pip install 'hypercli-cli[comfyui]'`
- local transcription: `pip install 'hypercli-cli[stt]'`

## Configure The Main API Key

```bash
hyper configure
```

Test it:

```bash
hyper billing balance
hyper instances list
```

## Launch A GPU Job

```bash
hyper instances launch nvidia/cuda:12.6.3-runtime-ubuntu22.04 \
  -g l40s \
  -c "nvidia-smi"
```

Then inspect it:

```bash
hyper jobs list
hyper jobs logs <job_id> -f
hyper jobs shell <job_id>
```

## Run A Managed Flow

```bash
hyper flow text-to-image "a mountain lake at dawn"
hyper flow text-to-video "a slow cinematic flyover of a glacier"
```

## Use x402 Pay-Per-Use

Create and fund a wallet first:

```bash
hyper wallet create
hyper wallet address
```

Then run a pay-per-use flow:

```bash
hyper flow text-to-image "a mountain lake at dawn" --x402
```

## Set Up HyperClaw

```bash
hyper agent plans
hyper agent subscribe 1aiu
hyper agent config env
```

## Launch An OpenClaw Agent

```bash
hyper agents create --name my-agent --size medium
hyper agents list
hyper agents shell <agent_id>
```
