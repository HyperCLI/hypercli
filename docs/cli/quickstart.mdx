---
title: CLI Quickstart
description: Install the CLI, configure credentials, launch jobs, subscribe to HyperClaw, create an OpenClaw agent, and connect to it.
---

## Install

```bash
pip install hypercli-cli
```

Extras:

- wallet and x402: `pip install 'hypercli-cli[wallet]'`
- ComfyUI: `pip install 'hypercli-cli[comfyui]'`
- local speech-to-text: `pip install 'hypercli-cli[stt]'`

## Configure Platform Access

```bash
hyper configure
```

Verify:

```bash
hyper billing balance
hyper instances list
```

## Launch A GPU Job

```bash
hyper instances launch nvidia/cuda:12.6.3-runtime-ubuntu22.04 \
  -g l40s \
  -c "nvidia-smi"
```

Then inspect it:

```bash
hyper jobs list
hyper jobs logs <job_id> -f
hyper jobs shell <job_id>
```

## Subscribe To HyperClaw

```bash
hyper agent plans
hyper agent subscribe 1aiu
hyper agent status
```

## Generate Provider Config

```bash
hyper config openclaw
hyper config openclaw --apply
```

`hyper config openclaw --apply` writes `~/.openclaw/openclaw.json` and
sets `hyperclaw/kimi-k2.5` as the primary coding model.

## Create An Agent

```bash
hyper agents create --name docs-demo --size medium --wait
```

Current defaults:

- gateway host: `https://docs-demo.hyperclaw.app`
- desktop host: `https://desktop-docs-demo.hyperclaw.app`

## Connect To The Agent

Shell and logs are backend WebSocket flows:

```bash
hyper agents shell docs-demo
hyper agents logs docs-demo --ws
```

Gateway operations use the root host:

```bash
hyper agents config docs-demo
hyper agents gateway-chat docs-demo "Reply with exactly: hello"
```

Desktop opens at the desktop route:

```bash
hyper agents status docs-demo
```

Use the printed `Desktop` URL in a browser.
